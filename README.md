# Vidhai Sales Intelligence

AI-powered sales tool for generating customized capability decks and pitch scripts.

## Features

- 📊 **Capability Deck Generator** - Create customized presentations from your case study library
- 📞 **Multi-Stakeholder Pitches** - Generate role-specific cold call scripts
- 📚 **Smart Matching** - AI surfaces most relevant case studies
- ⚡ **Instant Export** - Download as PowerPoint, PDF, or copy text
- 📈 **History Tracking** - Access all past generations

## Tech Stack

- Pure HTML/CSS/JavaScript
- Supabase (Database + Auth)
- Vercel (Hosting)

## Setup

1. Create Supabase project
2. Run SQL setup scripts
3. Add environment variables to Vercel
4. Deploy
